Privacy Policy

This Privacy Policy explains what data SSupport Bot ("the Bot") collects, how it is used, and your rights.

1. Data Collected

  • Server Data: Guild IDs, configuration settings (ticket categories, welcome messages, blacklisted words, join roles).
  • User Data: Only Discord IDs when needed for moderation features (e.g., assigning roles, tickets).
  • OAuth Data: Basic Discord profile info (username, ID) when you log in to the dashboard.

2. How We Use Data

  • To provide Bot features (tickets, moderation, welcome messages, dashboard).
  • To save and restore server configuration settings.
  • To authenticate you on the dashboard via Discord OAuth.

3. Data Storage

Data is stored locally on the server hosting the Bot (e.g., configuration files). No data is sold, rented, or shared with third parties.

4. Data Removal

You can request deletion of your server's configuration data by removing the Bot from your server and contacting the developer.

5. Third Parties

The Bot interacts only with the Discord API and does not share data with other services.

6. Changes

This Privacy Policy may be updated at any time. Continued use of the Bot means you accept the updated policy.

If you have questions, contact "sjkgoat" on discord.
